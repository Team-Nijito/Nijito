Detailed Description

This is in charge of creating and supplying puppets for the command system.

Though puppets are fairly self-sufficient, they are not able to create themselves.

This looks for puppets in the Resources/Puppets folder.

Puppet Dialogue.VN.PuppetMaster.GetPuppet ( string  characterName)

Gets the puppet which matches the given name.

If the puppet does not exist, it will be created using whatever preset seems to best match based on the name. If it can't figure out which puppet to use, it will create a generic one. This generic puppet will still behave correctly, but it will not look correct. When generic puppet is used, a warning is printed in the debug console.

Parameters
characterNameName of the character we are making a puppit for. This is case insensitive.
Returns
The puppet, whether newly created or previously existing.
